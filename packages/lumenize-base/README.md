# @lumenize/lumenize-base

A de✨light✨ful base class for Cloudflare Durable Objects with NADIS auto-injection.

For complete documentation, visit **[https://lumenize.com/docs/lumenize-base](https://lumenize.com/docs/lumenize-base)**

## Features

- **Auto-injection**: Import NADIS packages and access via `this.svc.*` - no manual wiring
- **Type-safe**: Full TypeScript autocomplete for all registered services
- **Lazy loading**: Services instantiated only when accessed
- **Dependency resolution**: Stateful services automatically get their dependencies
- **Zero boilerplate**: Just extend LumenizeBase and go

## Installation

```bash
npm install @lumenize/lumenize-base
```
