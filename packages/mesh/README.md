# @lumenize/mesh

Lumenize Mesh - communication framework for Cloudflare Durable Objects.

For complete documentation, visit **[https://lumenize.com/docs/lumenize-mesh](https://lumenize.com/docs/lumenize-mesh)**

## Features

- **Mesh communication**: DOs, Workers, and browser clients as true peers
- **Type-safe RPC**: Operation chaining and nesting with TypeScript support
- **Auto-injection**: Import NADIS packages and access via `this.svc.*`
- **Identity propagation**: Automatic `callContext` with origin, caller, and auth
- **Lazy loading**: Services instantiated only when accessed

## Installation

```bash
npm install @lumenize/mesh
```
