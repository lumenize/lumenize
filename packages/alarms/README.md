# @lumenize/alarms

A de✨light✨ful alarm scheduling system for Cloudflare Durable Objects with cron support.

For complete documentation, visit **[https://lumenize.com/docs/alarms](https://lumenize.com/docs/alarms)**

## Features

- **One-time alarms**: Schedule tasks for specific dates/times
- **Delayed alarms**: Schedule tasks with second-based delays
- **Cron scheduling**: Recurring tasks with cron expressions
- **Persistent storage**: Survives DO eviction and restart
- **NADIS compatible**: Use standalone or with LumenizeBase auto-injection

## Installation

```bash
npm install @lumenize/alarms
```
